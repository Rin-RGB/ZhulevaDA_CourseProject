## Установка и запуск системы

Для развёртывания системы необходимо клонировать репозиторий с помощью команды:

```
git clone https://github.com/Rin-RGB/ZhulevaDA_CourseProject.git
```

Далее следует перейти в директорию проекта и установить зависимости серверной части:

```
cd back
npm install
node server.js
```

После запуска серверной части необходимо открыть новый терминал, не останавливая работу сервера, и выполнить установку зависимостей клиентской части и запуск frontend-приложения:

```
cd front
npm install
npm run dev
```

После успешного запуска система будет доступна в браузере по адресу:

```
http://localhost:8000/login
```

При первом запуске открывается страница авторизации. Для входа в систему с правами администратора (роль CEO) используются следующие учётные данные:
